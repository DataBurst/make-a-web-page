Lesson 2: Creating a Structured Document with HTML

HTML - HYPERTEXT MARKUP LANGUAGE - Standard markup language used to create web pages.

CSS - CASCADING STYLE SHEETS - Style sheet lauguage used for describing the look and formatting of a document written in a markup laguange.

DOM - DOCUMENT OBJECT MODEL - A cross-platform and language-indpendent convention for representing and interacting with objects in HTML (and other markup languages). The nodes of every document are organized in a tree structure, called the DOM TREE.
